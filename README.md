# SOC-WEE-CEE-PEE

This repository contains my work for the WEE-CEE-PEE SOC program. The topics were studied primarily from the *Competitive Programmer's Handbook*, and solutions to the problems I practiced are available in the **Problems Practiced** folder.

## Topics Covered

### Week 1

* Introduction to Competitive Programming
* Time Complexity Analysis
* Sorting Algorithms
* Data Structures
* Complete Search
* Range Queries

### Weeks 2–3

* Greedy Algorithms
* Dynamic Programming
* Amortized Analysis

### Week 4

* Graph Algorithms
